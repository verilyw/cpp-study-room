
+ `#`将符号转换成字符串形式
+ `##`连接两个符号

```c++
#define STR2(x) #x
#define STR(x) STR2(x)  // 这样二次计算的目的是，比如我x是个表达式求值，我需要得到这个值的字符串，而不是表达式的字符串

#define CAT(x, y) x##y
```

使用惯例。
+ `do while`大法
+ 二次求值
+ 可变长度的参数

```c
#define PRINT(x, ...) do {\
	printf(__FILE_NAME__ ":" STR(__LINE__) ": " x __VA__OPT__(,) __VA__ARGS__);\
} while(0)
```

```c
#define MIN(x, y) ({ \
	typeof(x) __x = (x); \
	typeof(y) __y = (y); \
	__x < __y ? __x : __y; \
})
```
